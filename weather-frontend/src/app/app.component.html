About Weather Monitoring System

  • The Weather Monitoring System is designed to provide real-time weather insights by retrieving data from the OpenWeatherMap API.
  • It offers daily summaries and alerts based on user-defined weather thresholds.
  • Future updates will include real-time data integration and predictive analytics.
Objective

Develop a real-time data processing system to monitor weather conditions and provide summarized insights using rollups and aggregates from the OpenWeatherMap API.

Data Source

The system retrieves weather data from the OpenWeatherMap API at configurable intervals for six metros in India (Delhi, Mumbai, Chennai, Bangalore, Kolkata, Hyderabad).

  • Weather Parameters: Main weather condition, current temperature (Celsius), perceived temperature, and time of update (Unix timestamp).
Processing & Analysis
  • Real-time Data: Fetches data every 5 minutes, converting temperature from Kelvin to Celsius.
  • Daily Weather Summary: Aggregates average, max, and min temperature data and stores daily summaries in persistent storage.
  • Alerting Thresholds: Configurable thresholds allow users to trigger alerts based on temperature or weather conditions.
  • Visualization: Displays daily weather summaries and historical trends.
Web Technologies Used:
Angular TypeScript Bootstrap OpenWeatherMap API
